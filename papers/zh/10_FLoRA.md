# FLoRA：基于异构低秩适配的大语言模型联邦微调

**论文：** [arXiv 2409.05976](https://arxiv.org/abs/2409.05976)  
**发表于：** NeurIPS 2024  
**年份：** 2024  
**关键词：** 联邦 LoRA、聚合噪声、堆叠聚合、异构 LoRA、NeurIPS

---

## 问题背景

主流联邦 LoRA 微调方法将 FedAvg 直接应用于 LoRA 矩阵 A 和 B。这在**数学上是错误的**：由于 LLM 的权重更新为 ΔW = B·A，先对矩阵取平均再相乘 ≠ 先相乘再取平均。这引入了**聚合噪声**，降低微调效果，且无法处理不同客户端持有不同秩 LoRA（形状不同矩阵无法直接平均）的情况。

## 核心思路

FLoRA 提出了一种**基于堆叠的聚合策略**，从根本上规避聚合噪声。服务器不对客户端 LoRA 矩阵取平均，而是将所有客户端的 A 和 B 矩阵**堆叠**在一起，形成高秩的组合适配器。由堆叠矩阵计算出的 LLM 权重更新在数学上等价于每个客户端更新的总和——从构造上保证无噪声。

## 方法

- 每个客户端在本地训练自己的 LoRA 适配器 (A_i, B_i)。
- 服务器接收所有 (A_i, B_i) 对并进行堆叠：A_stack = [A_1; A_2; ...; A_n]，B_stack = [B_1, B_2, ..., B_n]。
- 堆叠后的适配器等价于 Σ B_i · A_i，在无平均噪声的情况下匹配理想聚合更新。
- 异构秩被自然处理：堆叠操作不受各客户端秩差异的影响。
- 可选通过奇异值分解（SVD）将堆叠适配器压缩回目标秩，再分发给客户端。

## 实验结果

- 在同构和异构秩设置下均显著优于 FedAvg + LoRA。
- 无噪声聚合加速收敛并提升最终准确率。
- 可扩展至大量客户端；在多样 NLP 基准上得到验证。
- 发表于 NeurIPS 2024，是该工作影响力最高的发表平台之一。

## 优点

- 修复了几乎所有已有联邦 LoRA 方法中存在的根本性数学错误。
- 无噪声特性可从理论上证明，而非仅仅来自实验观察。
- 无缝处理异构 LoRA 秩——无需秩归一化。
- 可直接替代任何联邦 LoRA 流程中的 FedAvg，无需额外改动。

## 局限性

- 堆叠聚合使服务器内存随客户端数量线性增长（需同时持有所有适配器）。
- 基于 SVD 的秩压缩增加了服务器端计算开销。
- 不显式处理个性化——全局模型质量提升，但每个客户端的个性化适配是次要效果。

## 与个性化联邦学习的关联

FLoRA 提升了联邦全局模型的质量，这是任何**两阶段流程**（全局联邦预训练 → 本地个性化）中个性化微调的基础。全局模型越好，个性化微调的起点越优，最终个性化效果也越强。
